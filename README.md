## ⚙️ 실행 방법

1. Node.js 설치를 확인해주세요.
2. 프로젝트를 클론 합니다.

```bash
  git clone https://github.com/soyoung125/wanted-pre-onboarding-frontend.git
```

3. 프로젝트에 진입합니다.

```bash
  cd wanted-pre-onboarding-frontend
```

4. 패키지를 설치합니다.

```bash
  npm install
```

5. 서버를 실행합니다.

```bash
  npm start
```
